Cells accept [hb:0.75] harvey balls, [up]/[down]/[flat] arrows, [good]/[bad] colors.

Agenda

One chapter per line. Inserts one agenda slide per chapter (appended at the end of the deck — move them into place), each with its chapter highlighted.

Automation (JSON)

Export the current chart as JSON, edit or generate it programmatically, and import it back — or paste an array of configs and insert them all. Same schema as npm run render.

Testing

Drop a demo of every chart kind plus key features and elements onto fresh slides (appended at the end of the deck) for quick visual testing. Each chart stays editable — delete the slides when you're done.
